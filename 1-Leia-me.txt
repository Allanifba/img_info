Para executar em Python (Recomendo PyCharm): 
(1) Crie um novo projeto.
(2) Coloque na pasta principal do projeto os arquivos img_info.py, 0.png e 0.txt.
(3) No terminal digite pip install PySimpleGUI
(4) Salve na mesma pasta que o arquivo img_info as imagens e o txts de mesmo nome:
Exemplo: nome.png e nome.txt
RECOMENDA-SE USAR IMAGENS DE MESMO TAMANHO QUE 0.png 
(Tamanho padrão = 182x250 pixels)
(5) Execute img_info.py
(6) Em um dos 10 campos de entrada digite nome e aperte o botão Selecionar. Para acessar as
informações contidas no arquivo nome.txt clique na imagem que irá aparecer.

Para criar um executável (Recomendo o PyCharm):
(1) No terminal digite pip install PySimpleGUI (caso não tenha feito anteriormente)
(2) No terminal digite pip install pyinstaller
(3) No terminal digite pyinstaller --onefile -w img_info.py
(4) O arquivo img_info.exe estará na pasta dist.
(5) Salve este arquivo na mesma pasta das imagens e dos txts mesmo nome:
Exemplo: nome.png e txt.png
RECOMENDA-SE USAR IMAGENS DE MESMO TAMANHO QUE 0.png 
(Tamanho padrão = 182x250 pixels)
(6) Execute o arquivo img_info.exe
(7) Em um dos 10 campos de entrada digite nome e aperte o botão Selecionar. Para acessar as
informações contidas no arquivo nome.txt clique na imagem que irá aparecer.
